iTerm commands for git, docker, gantry, grunt, etc.

Terminal

Command Does Notes
mkdir [folder name] make a directory
touch [file name] Create a file

Git

Command Does Notes
atom .gitignore Open gitignore Tell git which files to ignore
git rm -f folder/..../.DS_Store' Remove a file from git tracking git rm [-f | --force] [-n] [-r] [--cached] [--ignore-unmatch] [--quiet] [--] …​

Docker

Command Does Notes
docker ps list open docker containers too many containers slows your machine down - housekeeping
docker stop [CONTAINER NAME] Stop a container by sending SIGTERM and then SIGKILL after a grace period https://docs.docker.com/engine/reference/commandline/stop/
docker-machine start starts docker Started machines may have new IP addresses. You may need to re-run the `docker-machine env` command.
docker-machine env tell Docker to talk to the new machine https://docs.docker.com/machine/get-started/
eval $(docker-machine env) configure your shell https://docs.docker.com/machine/get-started/
